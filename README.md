# Django-Template-Generator-Linux-

Creates the following items as follows:
1. The Folder where all your files and folder will be stored
1.1 Checks if there is a folder with the same name
2. Your Virtual Environment
3. The Project Folder
4. The Application Folder
4.1 Adds the application in the INSTALLED APPS in the Project's settings.py
4.2 Creates the urls.py for the application folder
4.3 Adds the Application Url in the Project's URL
5. Opens your Project in VSCode

Limitations
1. If you have another folder with the same name in the beginning of the program, the program will terminate and you need to start it again.
2. Cannot install multiple application.
3. If you dont have VSCode installed. well, i dont know
